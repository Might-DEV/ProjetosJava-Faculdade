## Projeto: Relógio Digital em Java Swing ##
Um relógio digital simples que permite definir a hora inicial no código e exibe a contagem em tempo real.

### ⚙️ Funcionalidades
- Definição da hora inicial
- Atualização dinâmica dos segundos
- Interface gráfica com `JFrame` e `JLabel`
- Uso de Threads para manter a interface responsiva
